## How to Take Screenshots

1. Log in as a chef account
2. Navigate to each section using the sidebar
3. Take a screenshot of the page at a reasonable viewport size (1280x800 recommended)
4. Name the file exactly as shown above
5. Save as PNG format
6. Place in this directory

## Notes
- Include some sample data where possible to show realistic usage
- Capture the full page content, not just partial views
- Ensure the sidebar is visible in navigation screenshots
- Use consistent browser zoom level for all screenshots
